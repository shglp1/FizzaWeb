# FIZZA Production Foundation

Next.js 15 + TypeScript + Tailwind + Supabase architecture for scheduled family transportation.

## Run
1. Copy `.env.example` to `.env.local`
2. Fill Supabase and integration keys
3. `npm install`
4. `npm run dev`

## Included
- App Router routes for all requested modules
- Supabase SQL migration with core tables, enums, FK constraints, and seed data
- RLS starter policies
- Role-ready domain types
- Auth service and rider service
- React Hook Form + Zod validation
- Responsive layout (sidebar + mobile bottom nav)
- PWA manifest + installable metadata
